<h3 class="my-4">Why customElement()?</h3>
customElement() was designed primarily to speed up the creation of custom elements.
Unlike others, Lit, Stencil, or any, this only uses custom elements to make the 
coding structure as simple as possible without relying on Shadow DOM.
